usage: java -jar reverseFLHook-<version>.jar <command> <arguments> [<flags>]

# Commands
    survey   <system> <base>; Generates hyperspace coordinates (MK3 accuracy by default, valid for [28, 35] days)
    refresh  <coords>; Generates new hyperspace coordinates with the same location and accuracy as the ones provided (valid for [28, 35] days)

# Arguments
    system   System nickname, e.g. 'ew01' for Tau-37
    base     Base nickname, e.g. 'ew01_01_base' for Freeport-10
    coords   Full hyperspace coordinates string, e.g. '16FE3679-B69CDBB3-341841A5-C6AD29C4-DAFAA176-FD8A5201-2DFD9B72'

# Flags
    --admin  Uses admin-level accuracy when generating hyperspace coordinates
    --copy   Copies the generated hyperspace coordinates to the clipboard
    --auto   After a 5s delay, types '{ENTER}/setcoords <coords>{ENTER}' as if keys were pressed on the keyboard

# Examples
    java -jar reverseFLHook-<version>.jar survey ew01 ew01_01_base --auto
    java -jar reverseFLHook-<version>.jar refresh 16FE3679-B69CDBB3-341841A5-C6AD29C4-DAFAA176-FD8A5201-2DFD9B72 --copy
